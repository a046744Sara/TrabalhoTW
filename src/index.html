Bem-vindo

Este website foi desenvolvido no âmbito da disciplina de Tecnologias Web. O objetivo é demonstrar como a comunicação visual e os recursos multimédia ajudam a construir a identidade de uma marca no contexto digital.

Ao longo das quatro páginas, apresentamos os principais elementos de identidade visual, a importância da estratégia (objetivos e público-alvo) e o papel do multimédia na criação de conteúdos para websites e plataformas digitais.

A forma como uma marca usa cor, tipografia e composição influencia a perceção do público. Navega pelo menu para descobrires mais sobre os elementos da identidade visual.

Contexto académico: este site foi desenvolvido para fins educativos, aplicando requisitos de HTML e CSS. Os exemplos são demonstrativos e não representam uma marca real.

Recurso útil: Validador W3C (HTML) — ajuda a verificar se o teu código está válido e bem estruturado.

O que vais encontrar

  • Identidade Visual: elementos (logótipo, tipografia, cor) e coerência.
  • Estratégia: objetivos, público e consistência na comunicação.
  • Multimédia: formatos e dados estruturados para sites.

Este parágrafo está escondido (exemplo de requisito CSS).
